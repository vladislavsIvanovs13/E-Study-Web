# E-Study-Web
Web diary for both students and professors.

Running backend
===========================================
To start backend run this command:

```cd backend```

Then start _EStudyApplication.java_.

Do not forget to import _pom.xml_ dependencies.

Running frontend
============================================
To start frontend run these commands:

```
cd frontend
npm install
npx vite
```
